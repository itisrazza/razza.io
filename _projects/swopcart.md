Swopcart is a personal video game library manager inspired by [Jellyfin].

[Jellyfin]: https://jellyfin.org/

My goal for this project is to build a central library for the titles I own
but not part of a store ecosystem (i.e. Steam), and allow to install and
uninstall them as needed without losing my save data.

I'm slowly putting the pieces together over in the following GitHub org:
[https://github.com/swopcart](https://github.com/swopcart)
